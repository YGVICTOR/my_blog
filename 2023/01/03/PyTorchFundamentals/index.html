Creating tensors

scalar

A scalar is a single number and in tensor-speak it’s a zero dimension tensor.

vector

A vector is a single dimension tensor containing many numbers.

ndim is the number of square brackets on the outside([) and only one side is counted.

matrix

Name what is it Number of dimensions Lower of upper
scalar a single number 0 Lower(a)
vector numbers with direction 1 Lower(y)
matrix a 2-dimensional array of numbers 2 Upper(Q)
tensor n-dimensional array of numbers 0-n Upper(X)
